ForLab+

ForLab+ is a standardized, open-source web based tool with clearly defined requirements, improving programs’ ability to collect and analyze data to accurately forecast commodity needs. ForLab+ performs long-and short-term forecasts and guides improvements in diagnostic services.

Note: User can download the window version of ForLab 2.0 from this page using the download button below.
ForLab+, which is the web based version of the tool is in testing phase and is not ready for use at the moment. We will notify user when the tool is ready.
